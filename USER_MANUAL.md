## Installation

### System Requirements
- Android 5.0 (API 21) or higher
- At least 50 MB of free storage
- Microphone (for voice features)
- Camera (optional, for photo attachments)

### Installation Steps

1. **Download the APK**
   - Download `flutter_tasker_pro.apk` from the releases page
   - Or scan the QR code [if provided]

2. **Install the App**
   - Open the APK file on your Android device
   - If prompted, allow installation from unknown sources
   - Tap "Install" and wait for completion
   - Tap "Open" to launch the app

3. **Grant Permissions**
   - The app will request necessary permissions:
     - **Microphone** - For voice input and voice notes
     - **Storage** - For saving attachments and exports
     - **Camera** - For photo attachments
     - **Biometric** - For fingerprint/face unlock
   - Grant all permissions for full functionality

## Task Management

### Creating a Task

1. **Tap the + Button**
   - Located at bottom right of home screen

2. **Enter Task Details**

   **Title** (Required)
   - Type the task name OR
   - Tap üé§ microphone icon and speak
   - Example: "Buy groceries"

   **Description** (Optional)
   - Add more details OR
   - Use üé§ voice input
   - Example: "Get milk, eggs, bread from supermarket"

   **Due Date** (Required)
   - Tap the calendar icon
   - Select the date
   - Default is today

   **Due Time** (Optional)
   - Tap the clock icon
   - Set specific time
   - Leave empty for all-day task

   **Priority**
   - Choose Low (blue), Medium (orange), or High (red)
   - High priority tasks show a flag icon

   **Category** (Optional)
   - Select from dropdown
   - Categories help organize tasks
   - Create new categories in Settings

3. **Add Subtasks** (Optional)
   - Type subtask title OR use üé§ voice
   - Tap + button or press Enter
   - Add as many as needed
   - Each subtask has its own checkbox

4. **Add Attachments** (Optional)
   - Tap "Add Attachment" button
   - Choose:
     - **Take Photo** - Use camera
     - **Choose from Gallery** - Select existing photo
     - **Record Voice Note** - Record audio message

5. **Set Reminder** (Optional)
   - Tap "Reminder" dropdown
   - Select:
     - No reminder
     - 5, 10, 15, 30 minutes before
     - 1, 2, 24 hours before
   - You'll get a notification at that time

6. **Repeating Task** (Optional)
   - Turn on "Repeating Task" switch
   - Choose repeat type:
     - **Daily** - Every day
     - **Weekly** - Select specific days
     - **Custom** - Custom pattern
   - For weekly, select days (Mon, Tue, Wed, etc.)

7. **Save the Task**
   - Tap the checkmark ‚úì icon at top
   - Task appears in your list

### Viewing Task Details

1. **Tap any task card** in your list
2. You'll see:
   - Full title and description
   - Due date and time
   - Priority level
   - Category (if assigned)
   - Repeat settings (if repeating)
   - Completion percentage (if has subtasks)

3. **Subtasks Section**
   - List of all subtasks
   - Tap checkbox to mark complete
   - Swipe to delete
   - Add new subtasks

4. **Attachments Section**
   - Photo thumbnails (tap to view full size)
   - Voice notes with play button (tap to play/stop)

5. **Comments Section**
   - Read all comments
   - Add new comment
   - Delete your comments

### Editing a Task

1. Open task details
2. Tap the **Edit** icon (pencil) at top
3. Modify any field
4. Tap checkmark ‚úì to save changes

### Completing a Task

**Method 1: From Task List**
- Tap the checkbox on the left of task card
- Task moves to "Completed" tab

**Method 2: From Task Details**
- Open the task
- Tap checkbox
- Task is marked complete with timestamp

## Voice Features

### Voice Input for Text Fields

All text input fields support voice:

1. **Tap the üé§ microphone icon** next to the field
2. **Icon turns RED** - now listening
3. **Speak clearly** into your phone
4. **Speech converts to text** automatically
5. **Stops after 3 seconds** of silence

**Supported Fields:**
- Task Title
- Task Description
- Subtask Title

**Tips for Best Results:**
- Speak in a quiet environment
- Hold phone 6-12 inches from mouth
- Speak at normal conversational speed
- Pronounce words clearly
- Pause briefly between sentences

### Recording Voice Notes

Voice notes are attached to tasks like audio memos:

1. **In Add/Edit Task** screen
2. Tap **"Add Attachment"** button
3. Select **"Record voice note"**
4. Recording dialog appears with red microphone icon
5. **Speak your message**
   - No time limit
   - Icon shows recording in progress
6. **Tap "Stop Recording"** when done
7. Voice note is saved
8. Appears in task details as audio attachment

### Playing Voice Notes

1. **Open task details**
2. **Scroll to Attachments** section
3. **Tap voice note card** (shows play icon)
4. Audio plays
5. **Tap again to stop**

**Voice Note Tips:**
- Keep recordings under 2 minutes
- Speak clearly for better playback
- Use for quick reminders or thoughts
- Great for capturing ideas on the go

## Troubleshooting

### Voice Input Not Working

**Problem:** Microphone icon doesn't respond or no speech recognition

**Solutions:**
1. Grant microphone permission in device settings
2. Check if Google app is updated (voice recognition dependency)
3. Ensure internet connection (first-time setup)
4. Try restarting the app
5. Check device compatibility (Android 5.0+)

### Voice Notes Won't Record

**Problem:** Recording fails or doesn't save

**Solutions:**
1. Grant microphone permission
2. Check storage space (need space for audio files)
3. Grant storage permission
4. Restart app
5. Try recording shorter clips

### Audio Playback Issues

**Problem:** Voice notes won't play or no sound

**Solutions:**
1. Check device volume (turn up media volume)
2. Ensure file exists (not deleted from storage)
3. Try restarting app
4. Check if headphones/Bluetooth connected
5. Test with device music player

### Notifications Not Appearing

**Problem:** Not receiving task reminders

**Solutions:**
1. Grant notification permission
2. Check notification settings in device
3. Ensure "Do Not Disturb" is off
4. Battery optimization: Disable for this app
5. Test with "Test Notification" in Settings

### App Crashes or Freezes

**Problem:** App closes unexpectedly or becomes unresponsive

**Solutions:**
1. Restart the app
2. Clear app cache (device settings ‚Üí Apps ‚Üí Tasker Pro ‚Üí Clear Cache)
3. Ensure device has free RAM
4. Update to latest version
5. Reinstall app (backup data first!)

### Can't Login

**Problem:** Login fails with correct credentials

**Solutions:**
1. Check email spelling
2. Verify password (case-sensitive)
3. Try "Forgot Password" (if feature available)
4. Clear app data and start fresh
5. Check database isn't corrupted

### Export Files Not Found

**Problem:** Can't find exported files

**Solutions:**
1. Check Downloads folder using file manager
2. Grant storage permission
3. Try exporting again
4. Use different format (JSON instead of PDF)
5. Check device storage space

### Import Fails

**Problem:** JSON import doesn't work

**Solutions:**
1. Verify file is valid JSON (open in text editor)
2. Ensure file created by this app
3. Check file isn't corrupted
4. Try smaller backup file
5. Contact support with error details

## FAQ

### General Questions

**Q: Is Flutter Tasker Pro free?**  
A: Yes, the app is completely free to use with all features unlocked.

**Q: Does the app require internet?**  
A: No, works fully offline. Internet only needed for voice recognition first-time setup.

**Q: Can I use the app on multiple devices?**  
A: Yes, but you need to manually export/import data. Cloud sync coming in future update.

**Q: How secure is my data?**  
A: Data stored locally on your device in encrypted SQLite database. Enable biometric lock for extra security.

**Q: Can I sync with Google Calendar?**  
A: Not yet. Calendar integration planned for future release.

**Q: How many tasks can I create?**  
A: Unlimited tasks, subtasks, and categories.

**Q: Can I share tasks with my team?**  
A: You can share task details as text. Real-time collaboration coming in future.

**Q: Can I back up to cloud?**
