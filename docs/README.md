# Flutter Tasker Pro - Documentation

This directory contains the web-based documentation for Flutter Tasker Pro.

## Files
- `index.html` - Main landing page
- `privacy-policy.html` - Privacy Policy
- `terms-and-conditions.html` - Terms & Conditions

## GitHub Pages Setup

These files are hosted via GitHub Pages. To access them:

1. Enable GitHub Pages in repository settings
2. Set source to `main` branch and `/docs` folder
3. Pages will be available at: `https://atif327.github.io/Tasker-pro/`

- Privacy Policy: https://atif327.github.io/Tasker-pro/privacy-policy.html
